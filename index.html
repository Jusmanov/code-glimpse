About codeGlimpse

What is codeGlimpse?

codeGlimpse is an anonymous code-sharing platform for coders where individuals can upload snippets of code they find useful and beneficial to the community. The idea behind this website is to help form valuable connections between coders. codeGlimpse is welcome to all, regardless of one's coding ability and experience. There is no limit to learning, helping, and innovating, and there never should be.

Who are the ingenious, brilliant mastermind virtuosos behind codeGlimpse?

codeGlimpse was fabricated by a one-man team. My name is John, but people know me as jusmanov online. I'm a rising junior in high school and am looking to broaden my horizons through coding I started coding about a year ago, and I've begun participating in hackathons and competitions just recently. I'm always looking for new challenges as well as ways to help others as much as I can, even with a limited skillset.

What are your future plans for codeGlimpse?

I made codeGlimpse as a small project looking to make a small positive impact, so scaling this website is not in my plans in the future unless the website attains real traffic from users. In that case, the website would be scaled and updated regularly.

    Beginner

    For the beginner-level fitness tracker challenge, participants will create a simple console-based application to track basic fitness activities. The program will prompt users to input their daily exercise activities, such as the number of steps taken, the duration of workouts, and the number of calories consumed. Participants will design the program to calculate and display relevant fitness metrics, such as total steps taken, average workout duration, and estimated calorie burn. This challenge will help beginners practice fundamental programming concepts like user input, data manipulation, and basic arithmetic operations.

    Intermediate

    In the intermediate challenge, participants will build a more comprehensive fitness tracker web application. They will utilize HTML, CSS, and JavaScript to create a user-friendly interface where users can log their fitness activities and view their progress over time. The application should include features like a calendar view to visualize daily activities, charts to display weekly or monthly trends, and the ability to set fitness goals. Participants will also implement basic user authentication to allow multiple users to track their individual fitness data. The challenge will require skills in frontend development, DOM manipulation, and using libraries for data visualization.

    Advanced

    The advanced-level fitness tracker challenge will involve developing a mobile app with advanced functionalities. Participants will build the app using a cross-platform framework like React Native or Flutter. The app should integrate with wearable fitness devices and sensors to automatically track activities like steps, heart rate, and sleep patterns. Additionally, participants will implement machine learning algorithms to provide personalized fitness recommendations based on users' activity data and goals. The app should have a robust backend that stores and synchronizes data securely across devices. This challenge will test participants' proficiency in mobile app development, API integration, machine learning, and handling real-time data.
